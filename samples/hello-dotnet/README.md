# hello-dotnet sample

Tiny .NET sample with one class and one passing xUnit test.

## What is inside

- src/HelloApp: `GreetingService` with one method.
- tests/HelloApp.Tests: one test that verifies the greeting output.

## Run it

From the repository root:

```bash
dotnet test samples/hello-dotnet/HelloDotnetSample.slnx
```
